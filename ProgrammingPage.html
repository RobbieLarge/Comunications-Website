Programming Languages

Some Of The Different Programming languages Used Today

  • Python: A high level interpreted programming language, known for its versatility and easy to read syntax. it is widely used in Web-Development, Artificial Intelligence, Data Analysis, Document Automation and much more.
  • Java: A class based object oriented language that is designed in a way that reduces the need for external dependencies when it is being implemented. It is wisely used in chat applications, back-end of applications, android app development, financial services
  • JavaScript: is a high level programming language that is used for its versatility in front end and back end web and application development, game development
  • C#/C Sharp: Is a more modern object oriented programming language appose to java. Developed by Microsoft C# is used mainly for developing desktop applications and web applications via asp.net framework and game development with unity
  • C++: Is and addition to the C language with more features such as classes, templates and virtual functions. It is used for system and software development, game development and applications where performance is important.
  • HTML: HTML stands for hyper text markup language. It is the standard language used for displaying text in a web browser.
  • CSS: CSS stands for cascading style sheets. This language is used for styling written text in a document and describes how it should be presented.
  • PHP: A general purpose scripting language suited for Web-Development. It can be embedded into html and is use full when building complex web applications
