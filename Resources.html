Online Resources To Learn How to code

There are many different free online resources available to learn how to program. w3schools can be a very good tool to learn how to code since it has a large variety of code available and indepth concept of each language. Hacker Rank is also a great way to practice your coding skills by solving coding problems, in any programming language you would like to improve on. There are different coding problems that can be solved for example there are algorithms, data structures and many more.

IDE / Applications For Coding Or Programming

There are a many different ides available to download for free for all operating systems to program/code. Ides make it easier to code by highlighting errors, highlighting syntax to make it easy to read , running the program in its own terminal or shell. Most ides alow you to download packages and extensions for different languages and features. Some ides that have a subscription will help solve bugs and rewrite small blocks of code.
