Possible career paths for programmers

  • Web Developer: Creating websites and or web applications using programming languages such as HTML,CSS and javascript
  • Mobile app developer: Developing applications for mobile devices, such as smart phones and tablets for platforms like IOS and Android
  • Cyber Security: protecting computer systems and networks from cyber threats by implementing security measures and protocols
